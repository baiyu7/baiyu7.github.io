5 break 和 continue

​ 

    break用于结束循环体

    //判断1-10000之间能被99整除的数是不是超过了20个
    public static void main(String[] args) {
    int count=0;
    for(int i=1;i<=10000;i++){
        if(i%99==0){
            count++;
            if(count>20){
                System.out.println("超过20个"+i);  
                break;  //结束当前循环体
            }
        }
    }
    if(count<20){
        System.out.println("没超过20个");
    }

}

//此例子演示break结束的离他最近的循环    
for(int i=0;i<5;i++){
        for(int j=0;j<10;j++){
            System.out.print(j+" ");
            if(j>6){
                break;
            }
        }
        System.out.println();
    }    

//标号的使用
public static void main(String[] args) {
    AAA:for(int i=0;i<5;i++){
        BBB:for(int j=0;j<10;j++){
            System.out.print(j+" ");
            if(j>6){
                break AAA;  //结束名字为AAA的循环体
            }
        }
        System.out.println();
    }

}

// continue的使用

for(int i=0;i<=20;i++){
        if(i%5!=0){
            continue;  //结束当前这层循环,继续下一层次的循环,循环体不会像break一样结束
        }
        System.out.println(i);
}

6 函数

​ 

什么是函数?
    函数就是定义在  类中   的具有  特定功能 的一段   独立的程序,函数也称为方法

函数的格式:

    修饰符  返回值类型 函数名称(参数类型 形式参数1,参数类型 形式参数2......){
        函数体
        return 返回值  //可以省略
    }

返回值类型:函数运行后的结果的数据数据类型,和返回值的类型有绝对关系
参数类型:是形式参数的数据类型
形式参数:是一个变量,用来存储调用函数时传递给函数的实际参数
实际参数:传递给函数的真实哪个参数
return: 结束函数体
返回值:改值会返回给调用者

返回值类型:有返回值  基本数据类型  引用数据类型
           无返回值  void

System.out.println(“请输入第二个数”);
int num2=scan.nextInt();
System.out.println(“请输入符号”);
String str=scan.next();
char ch=str.charAt(0);
int result=jiSuanQi(num1,num2,ch);
System.out.println(result);
}

public static int jiSuanQi(int num1,int num2,char ch){
    int result=0;
    switch (ch) { 
    case '+':
        result=num1+num2;
        break;
    case '-':
        result=num1-num2;
        break;
    case '*':
        result=num1*num2;
        break;
    case '/':
        result=num1/num2;
        break;
    default:
        System.out.println("您输入的符号有误");
        break;
  }
    return result;
}

//例子
public static void main(String[] args) {
    System.out.println("找一个会铺地板的师傅");
    int a=10;
    int b=20;
    铺地板(a,b);

    int money=计算工钱(a,b,7);
    System.out.println("最后钱数:"+money);

}

public static void 铺地板(int a,int b){
    for(int i=0;i<a;i++){
        for(int j=0;j<b;j++){
            System.out.print("#");
        }
        System.out.println();
    }
}

public static int 计算工钱(int a,int b,int money){
    return a*b*money;
}

7 重载函数

​ 

在一个类中,可以存在名称相同,但是函数的参数个数,或者参数类型不同的方法,这样的函数称为重载
函数重不重载,不看返回值类型

以下和  void  a(int a,char b,String c) 是重载

void a(int a,char y,String z)  //不是,和原函数一模一样
int a(int a,String b,char c);  //是重载
void a(int a,String b,String c)  //是重载
int a(int a,String b)  //是
int a()    //是
boolean a(int a,char b,String name)  //不是
